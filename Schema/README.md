# Schema

This folder contains a small set of starter JSON Schema files for SysForge.

## Files

- `user.schema.json` — validates user/account records.
- `project.schema.json` — validates project records.
- `task.schema.json` — validates task/work-item records.
- `index.schema.json` — simple collection-level schema that references the other schemas.

These schemas use JSON Schema Draft 2020-12 and can be expanded as the project grows.
